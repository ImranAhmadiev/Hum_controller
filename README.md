# Humidity controller
## Контроллер влажности почвы

Данный прибор регистрирует влажность почвы и отображает ее на веб-сервере.
## Компоненты:
- Плата NodeMCU (ESP8266)
- Емкостной датчик влажности почвы
- Светодиод 
- Резистор 150 Ом
- Комплект проводов
